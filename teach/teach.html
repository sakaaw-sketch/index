教學中心 - 點播與會員系統指南

點播系統教學

1. 登入會員
點擊 會員中心 並登入您的帳號。如果您尚未註冊，請參考 會員系統教學。

2. 進入點播系統
登入後，點擊會員中心儀表板中的「福利申請」卡片，然後選擇「點播系統」。

3. 搜索歌曲
在點播系統中，使用搜索功能輸入歌曲名稱或歌手名稱（建議使用日文）。您也可以瀏覽專輯或系列。

4. 申請點播
找到您想點播的歌曲後，點擊「申請點播」按鈕。點播成功後，歌曲將進入播放列表。


會員系統教學

1. 註冊會員
訪問 註冊頁面，填寫必要信息並提交申請。

2. 登入系統
註冊成功後，前往 登入頁面，輸入您的用戶名和密碼。

3. 訪問會員中心
登入後，您將被導向 會員中心，這裡可以管理您的點播申請、查看進度等。


找尋 YouTube ID 和 Discord ID 教學

1. 找尋 YouTube ID

YouTube ID 是您的 YouTube 頻道唯一識別碼（例如 UC_x5XG1OV2P6uZZ5FSM9Ttw），用於綁定會員帳戶。請按照以下步驟查找：

  • 開啟您的 YouTube 頻道頁面（例如 https://www.youtube.com/@YourChannelName）。
  • 在瀏覽器地址欄中，複製 URL 並檢查是否包含 channel/ 後的 ID（例如 https://www.youtube.com/channel/UC_x5XG1OV2P6uZZ5FSM9Ttw）。
  • 若 URL 未顯示 ID，右鍵點擊頻道名稱，選擇「檢視頁面原始碼」，搜尋 channelId，即可找到您的 ID。
  • 將 ID 複製並貼到會員中心的個人資料頁面。

提示：確保使用正確的頻道 ID，而非用戶名或自訂 URL。若無法找到，請聯繫 support@example.com。

2. 找尋 Discord ID

Discord ID 是您的 Discord 帳戶唯一識別碼（例如 123456789012345678），用於綁定會員帳戶。請按照以下步驟查找：

  • 開啟 Discord 應用程式，點擊左下角的「用戶設置」（齒輪圖標）。
  • 在「外觀」或「進階」設定中，啟用「開發者模式」。
  • 返回主介面，右鍵點擊您的用戶名（在伺服器或好友列表中），選擇「複製 ID」。
  • 將複製的 ID 貼到會員中心的個人資料頁面。

提示：Discord ID 是數字，而非您的用戶名或標籤。若無法啟用開發者模式，請參考 Discord 官方指南。
